# Partilha App

Organize, divida e acompanhe despesas em grupo de forma simples e colaborativa.

---

## Visão Geral

Aplicativo mobile desenvolvido com [Expo](https://expo.dev).

## Como começar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o app:
   ```bash
   npx expo start
   ```
